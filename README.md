### Signal

A Grade 10 student at Sabari Central School, Palakkad, working at the intersection of astrophysics, aerospace engineering, and machine learning. This repository index reflects work built independently — exoplanet detection pipelines, hardware security tooling, and a bilingual science-literacy platform for a district where those pathways are rarely made visible.

Palakkad is also home to the Kerala School of Mathematics, the medieval tradition that anticipated calculus centuries before Newton and Leibniz. That inheritance is the quiet premise behind everything here: rigorous work doesn't need a metropolitan address.
